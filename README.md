### üìå Description

**BOOST-EO** is a set of custom code `snippets` & `pattern` available for VScode IDE to help you be more productive by providing helper methods üöÄ.




---

### üìå Installation

You can install it by typing `boost-eo` in the extension tab of your IDE




---

### üìå Features

- [x] Javascript snippets collection
- [x] Javascript patterns collection
- [x] Some snippets on frequently used javascript libraries
- [ ] Typescript snippets collection
- [ ] Typescript patterns collection
- [ ] React snippets collection
- [ ] React patterns collection
- [ ] Python snippets collection
- [ ] Python patterns collection
- [ ] Shell script snippets collection
- [ ] Shell script patterns collection

### üìå Contribution

üèÅ Before contributing, be sure to `install all dependencies` needed to work in this project by running the following command (in root directory):
```bash
npm install
```
üèÅ With each new code snippet added, you need to update all `snippets` & `patterns` file and regenerate the documentation, all in one command (in root directory):
```bash
npm run build
```
üèÅ You can run a `grunt task` to make a `local backup` of this project by running the following command (in root directory):
```bash
grunt compress-all
```
üèÅ Here is a tool that will help you to generate snippets -> https://snippet-generator.app/

üèÅ Then you can make a PR.
